Project : mirrorITpc

My project, MirrorITpc, is a new technology that allows users to control their computer system using their mobile phone. By interfacing the computer hardware and software with the mobile phone, the computer can be switched on and off using the mobile phone, and some computer tasks can be performed using the same phone. The application turns your smartphone into a wireless keyboard and mouse with touchpad, and can be performed with a wireless connection between the PC or laptop and the smartphone with Android operating system. By accessing the IP address of the PC, a connection can be established between them using Wi-Fi connection. This application not only turns your smartphone into a wireless keyboard and mouse, but it also provides various other features including file transfer, file download, image viewer, presentation, media player, live screen, and power off. The implemented application consists of two parts: an application for Android smartphone and a jar application for Windows PC that executes the command selected by the userâ€™s application. The outcome of this implementation is a handy, easy-to-use application.
